## 損益分岐点とは

損益分岐点とは、売上高と総費用が等しくなる利益ゼロの売上高を指します。この売上高を超えれば利益が発生し、下回れば損失が出ます。ITパスポート試験では計算問題が定期的に出題される頻出論点です。

## 必須の用語

変動費は売上高に比例して増減する費用で、材料費や販売手数料が該当します。固定費は売上高に関わらず一定にかかる費用であり、家賃、人件費の基本給、減価償却費などが代表例です。限界利益は売上高から変動費を引いた金額で、固定費の回収と利益の源泉となります。限界利益率は限界利益を売上高で割った値であり、1から変動費率（変動費÷売上高）を引いたものと等しくなります。

## 損益分岐点の計算式

```
損益分岐点売上高 = 固定費 ÷ 限界利益率
```

または

```
損益分岐点売上高 = 固定費 ÷（1 − 変動費率）
```

## 計算例

売上高1,000万円、変動費600万円、固定費300万円の場合を考えます。変動費率は600÷1,000で0.6（60%）、限界利益率は1から0.6を引いて0.4（40%）です。これらを用いて損益分岐点売上高を求めると、固定費300万円を限界利益率0.4で割り、750万円となります。売上高750万円のとき利益はゼロであり、実際の売上高1,000万円のときの利益は100万円です。

## CVP分析とは

CVP分析は、費用（Cost）、販売量（Volume）、利益（Profit）の関係を分析する手法です。損益分岐点はその代表的成果物であり、この分析を用いることで、「いくら売れば利益が出るか」や「目標利益を達成する売上高はいくらか」を試算できます。

## 目標利益達成売上高

```
目標利益達成売上高 = (固定費 + 目標利益) ÷ 限界利益率
```

例えば、先ほどの例で目標利益200万円を達成するためには、(固定費300万円＋目標利益200万円)を限界利益率0.4で割った1,250万円の売上高が必要です。

## ITパスポート試験での出題ポイント

ITパスポート試験では、変動費・固定費・限界利益・限界利益率といった用語の識別が出題されます。また、損益分岐点売上高の計算問題や、目標利益達成売上高の応用計算も頻出です。

## 学習のコツ

学習のコツとして、まずは「固定費 ÷ 限界利益率」の公式を口で唱えて覚えましょう。また、限界利益率は1から変動費率を引いたものであることを即座に計算できるようにしておくと便利です。さらに、例題を2〜3問だけ手計算で解いておけば、本番で迷わずに解答できます。
