## 試験申込の基本情報

IT パスポート試験の申込は、IPA（情報処理推進機構）の公式サイト経由で行います。受験料は 2024 年改定で **7,500 円（税込）** です。支払方法はクレジットカード、コンビニ決済、ペイジー、バウチャー（学割など）に対応しています。試験は CBT 方式で通年実施されており、好きなタイミングで受験できます。

## 申込手順（5 ステップ）

### Step 1：利用者 ID を作成

まず IPA 公式サイトの「利用者メニュー」にアクセスし、メールアドレス・氏名・住所などを登録します。登録後、確認メールに記載された URL をクリックすると本登録が完了します。

### Step 2：プロメトリック ID を取得

IT パスポート CBT 試験はプロメトリック社が運営しており、利用者登録と同時に自動的に紐づく ID が発行されます。一度登録すれば次回以降も同じ ID を使い回せます。

### Step 3：試験日と会場を選ぶ

全国にあるプロメトリック認定会場から受験日と会場を選択します。都市部では土日が早く埋まる傾向があるため、平日や朝一の枠を狙うと空きが見つかりやすいです。月をまたぐと選択肢が広がります。

### Step 4：受験料を支払う

| 支払方法 | 注意点 |
| --- | --- |
| クレジットカード | 即時確定、最も簡単 |
| コンビニ決済 | 払込票が発行される |
| ペイジー | ネットバンキング・ATM |
| バウチャー | 学割や企業契約での前払い |

支払いが完了すると予約が確定します。

### Step 5：受験票（確認票）を取得

予約完了メールに確認票へのリンクが記載されています。印刷は不要で、メール画面を提示しても問題ありません。試験当日は本人確認書類と一緒に持参しましょう。

## 申込時の注意点

### 氏名・生年月日の正確性

受験票と本人確認書類の氏名・生年月日は完全一致している必要があります。旧姓を使用する場合や英字表記にずれがあると、受験できないケースもあるため注意してください。

### キャンセル・変更ルール

受験日の 3 日前までであれば手数料無料で変更・キャンセルできます。3 日以内になると手数料が発生するか、変更自体が不可になります。ルールは変更される可能性があるため、必ず最新情報を確認してください。

### 学割の活用

学校や教育機関を通じて受験料割引のバウチャーが入手できる場合があります。詳しくは進路担当に確認するか、[IT パスポート学割と受験料割引制度](/blog/exam-discount) を参照してください。

## 申込後〜試験当日

申込を済ませた時点から学習を開始すると、締切効果でモチベーションを維持しやすくなります。受験票は当日忘れずに持参しましょう。メールから再表示も可能です。当日の流れについては [CBT 試験当日の流れと持ち物](/blog/exam-day-cbt) で詳しく解説しています。

## よくある質問

### Q. 何回でも受験可能？

受験回数に制限はありませんが、毎回受験料が発生します。

### Q. 当日キャンセルは？

原則として当日キャンセルはできません。受験料は返金されません。

### Q. 受験票を紛失したら？

利用者メニューから再表示・再印刷が可能です。

### Q. 試験会場で何時に到着すべき？

試験開始の 30 分前到着を推奨します。
