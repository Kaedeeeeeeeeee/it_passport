## 什么是盈亏平衡点

盈亏平衡点是指销售额与总费用相等、利润为零的销售额。超过该销售额则产生利润，低于该销售额则出现亏损。在IT护照考试中，计算题是定期出现的常考知识点。

## 必备术语

变动费是随销售额成比例增减的费用，包括材料费和销售手续费等。固定费是不论销售额多少都固定发生的费用，典型例子有房租、基本工资、折旧费等。边际贡献是销售额减去变动费后的金额，用于回收固定费并产生利润。边际贡献率是边际贡献除以销售额的值，等于1减去变动费率（变动费÷销售额）。

## 盈亏平衡点的计算公式

```
盈亏平衡点销售额 = 固定费 ÷ 边际贡献率
```

或者

```
盈亏平衡点销售额 = 固定费 ÷（1 − 变动费率）
```

## 计算示例

假设销售额为1000万日元，变动费为600万日元，固定费为300万日元。变动费率为600÷1000=0.6（60%），边际贡献率为1减去0.6等于0.4（40%）。由此计算盈亏平衡点销售额，固定费300万日元除以边际贡献率0.4，得到750万日元。销售额为750万日元时利润为零，实际销售额为1000万日元时利润为100万日元。

## 什么是CVP分析

CVP分析是分析费用（Cost）、销售量（Volume）、利润（Profit）之间关系的方法。盈亏平衡点是其代表性成果，通过该分析可以估算“需要销售多少才能产生利润”或“达成目标利润所需的销售额”。

## 目标利润达成销售额

```
目标利润达成销售额 = (固定费 + 目标利润) ÷ 边际贡献率
```

例如，在上述示例中，要达成200万日元的目标利润，需要（固定费300万日元+目标利润200万日元）除以边际贡献率0.4，即1250万日元的销售额。

## 学习技巧

学习技巧方面，首先请反复背诵“固定费 ÷ 边际贡献率”这个公式。同时，能够迅速计算出边际贡献率等于1减去变动费率会很有帮助。此外，手动计算2~3道例题，就能在考试时毫不犹豫地作答。
